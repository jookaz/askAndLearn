# 智慧问学系统

* 问学：通过向大模型提问，并依据回复内容理解学习的以问促学过程。

* 智慧问学：通过你问你答、他问你答和协同改进，提升学生向大模型的提问能力，以及判别大模型回复的能力。

本系统使用SSM框架实现，使用Spring AI调用大语言模型。使用MySQL数据库存储数据。
